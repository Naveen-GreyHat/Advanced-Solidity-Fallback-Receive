# FallbackDemo - Advanced Solidity Fallback/Receive Demo

A comprehensive Foundry project demonstrating advanced usage of `receive()` and `fallback()` functions in Solidity with professional tooling and extensive testing.

## Features

- **Advanced Contract Features**: Ownership pattern, withdrawal security, contribution tracking
- **Comprehensive Testing**: Unit tests, fuzz tests, edge cases
- **Professional CLI**: Interactive menu with advanced options
- **Gas Optimization**: Gas reporting and optimization features
- **Deployment Management**: Automated deployment tracking
